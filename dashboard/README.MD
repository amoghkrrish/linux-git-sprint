# System Dashboard

A menu‑driven Bash script that integrates multiple system administration tools.

## Features
- **System Health** – uptime, memory, CPU, disk, top processes.
- **Log Analysis** – analyse Apache logs for top IPs and status codes.
- **Port Scanner** – scan a range of local ports.
- **Backup** – timestamped backups of chosen directories.
- **Disk Cleanup** – remove files older than N days with confirmation.

## Usage
```bash
./dashboard.sh

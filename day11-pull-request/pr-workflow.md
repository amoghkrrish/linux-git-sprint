# Pull Request Workflow (Day11)

1. Fork the repository on GitHub.
2. Clone your fork: `git clone git@github.com:<you>/<repo>.git`
3. Create a branch: `git checkout -b feature-branch`
4. Make changes, stage, commit.
5. Push branch to your fork: `git push origin feature-branch`
6. Open a Pull Request on GitHub from your branch to the original repo's main branch.
7. Discuss, review, merge.
8. Delete the branch (optional).
